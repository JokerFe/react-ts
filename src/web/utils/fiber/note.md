# fiber
- fiber:就是一个数据结构，它有很多属性，虚拟Dom是对真实dom的一种简化，一些真实DOM都做不到的事情，那虚拟DOM更做不到，所以就有了fiber
fiber有很多属性，希望借由fiber上的这堆属性，来做到一些比较厉害的事情
- fiber架构：
为了弥补一些不足，设计了一些新的算法，为了能让这些算法跑起来，所以出现了fiber这种数据结构。
fiber这种数据结构+新的算法=fiber架构
